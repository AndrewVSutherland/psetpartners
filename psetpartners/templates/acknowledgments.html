Software

The website runs on free open source software, including JavaScript, jQuery, Python, Flask, Jinja, Psycopg, PostgreSQL, Touchstone, and Git, and is partially derived from the open source projects Research Seminars and the LMFDB.

Hosting

The database and web server are hosted at the MIT mathematics department. The website source code is hosted on GitHub.

Administrators

The site is currently administered by David Roe and Andrew Sutherland.
