Contact information

First, please check the FAQ to see if your question or issue is addressed there.
  • If you have a question or need help using the site please email us; we will try to respond within 24 hours.

  • If you have a suggestion for how we can improve this site, please send a feature request.

  • To report a software error, please submit a bug report to psetpartners@mit.edu that includes your kerberos ID and a short description of the problem and how to reproduce it.
