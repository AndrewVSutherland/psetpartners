Frequently asked questions

  • Why is this FAQ so short?

    Not many questions have been asked yet. Ask one!

  • Who can use this site?

    This web site is accessible to all current MIT students via Touchstone authentication. For the fall 2020 semester only classes listed in course 18 are available on pset partners. Depending on how it goes this fall, we may expand in the spring.

  • Why are all the dorms and living groups grayed-out in the location drop down?

    Given that so many students will not be on campus this fall we didn't think it made sense to organize pset groups according to physical location. The "in-person" communication option is disabled for the same reason.

  • Which platforms do you support?

    We support recent versions of Brave, Chrome, Chromium, Edge, Firefox, Opera, and Safari running on most recent versions of Chrome OS, linux (tested on Pop! OS and Ubuntu), MacOS, and Windows, where recent means within the past few years. We have not had time to test every feature on every platform, so there may well be bugs. Please let us know if you find one.

    We do not support Internet Explorer (use Edge). We plan to add support for tablets running Android or iOS in the near future.
